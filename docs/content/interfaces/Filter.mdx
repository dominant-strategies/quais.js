title: Filter

A **Filter** allows searching a specific range of blocks for mathcing logs.

## Extends

- [`EventFilter`](/content/interfaces/EventFilter)

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `fromBlock?` | [`BlockTag`](/content/type-aliases/BlockTag) | The start block for the filter (inclusive). |
| `toBlock?` | [`BlockTag`](/content/type-aliases/BlockTag) | The end block for the filter (inclusive). |
