```ts
function copyRequest(req): PreparedTransactionRequest
```

Returns a copy of `req` with all properties coerced to their strict types.

| Parameter | Type | Description |

| `req` | [`TransactionRequest`](/content/type-aliases/TransactionRequest) | The transaction request to copy. |

## Returns

[`PreparedTransactionRequest`](/content/type-aliases/PreparedTransactionRequest)

The prepared transaction request.

## Throws

If the request is invalid.
