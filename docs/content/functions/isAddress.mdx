```ts
function isAddress(value): value is string
```

Returns true if `value` is a valid address.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | `any` | The value to check. |

## Returns

`value is string`

True if the value is a valid address.
