```ts
function getBigInt(value, name?): bigint
```

Gets a BigInt from `value`. If it is an invalid value for a BigInt, then an ArgumentError will be thrown for `name`.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | [`BigNumberish`](/sdk/content/type-aliases/BigNumberish) | The value to convert. |
| `name`? | `string` | The name of the value. |

## Returns

`bigint`

The value.
