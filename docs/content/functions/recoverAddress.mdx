```ts
function recoverAddress(digest, signature): string
```

Returns the recovered address for the private key that was used to sign `digest` that resulted in `signature`.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `digest` | [`BytesLike`](/sdk/content/type-aliases/BytesLike) | The digest of the message. |
| `signature` | [`SignatureLike`](/sdk/content/type-aliases/SignatureLike) | The signature. |

## Returns

`string`

The address.
