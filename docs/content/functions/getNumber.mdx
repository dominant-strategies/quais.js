```ts
function getNumber(value, name?): number
```

Gets a //number// from `value`. If it is an invalid value for a //number//, then an ArgumentError will be thrown for
`name`.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `value` | [`BigNumberish`](/sdk/content/type-aliases/BigNumberish) | The value to convert. |
| `name`? | `string` | The name of the value. |

## Returns

`number`

The value.

## Throws

If the value is invalid.

## Throws

If the value is too large.
