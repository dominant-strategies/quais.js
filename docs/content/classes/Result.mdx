A [**Result**](/content/classes/Result) is a sub-class of Array, which allows accessing any of its values either positionally by
its index or, if keys are provided by its name.

## Extends

- `Array`\<`any`\>

## Methods

### getValue()

```ts
getValue(name): any
```

Returns the value for `name`.

Since it is possible to have a key whose name conflicts with a method on a [**Result**](/content/classes/Result) or its
superclass Array, or any JavaScript keyword, this ensures all named values are still accessible by name.

#### Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `name` | `string` | The name of the value to retrieve. |

#### Returns

`any`

The value for `name`.

### toObject()

```ts
toObject(): Record<string, any>
```

Returns the Result as an Object with each name-value pair.

This will throw if any value is unnamed, or if there are any outstanding deferred errors.

#### Returns

### fromItems()

```ts
static fromItems(items, keys?): Result
```

Creates a new [**Result**](/content/classes/Result) for `items` with each entry also accessible by its corresponding name in
`keys`.

#### Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `items` | `any`[] | The items to include in the Result. |
| `keys`? | (`null` \| `string`)[] | The names for each item in `items`. |

#### Returns

[`Result`](/content/classes/Result)

The new Result.
